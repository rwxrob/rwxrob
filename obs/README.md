# Simple OBS Overlay

Vibe coded this to put logo and link in upper left, date and time in upper right, and current GitHub status in the middle, which is dynamically loaded every five minutes.

1. Update `index.html` to have your link
2. Update the image
3. Add a browser source in OBS and center
4. Start the `serve-status` server
5. Set GitHub status from web UI or use `ghstatus` command

I use this because I am frequently changing the title of the stream based on topic and make recordings to be uploaded.
